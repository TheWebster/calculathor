calculathor
===========

An expression parsing system for use in Notificathor.
Compiles the expression to opcode for later execution.

Done
----
* adding
* subtracting
* multiplying
* dividing
* brackets
* error handling
* precalculating stacksize
* other Datatypes (strings)
* cross referencing
